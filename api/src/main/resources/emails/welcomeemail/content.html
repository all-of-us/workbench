Helpful tips before you get started:
Chrome is the primary browser we currently support for the Researcher Workbench. We highly recommend using Chrome for the best user experience.
We recommend creating a Chrome Profile with your ${ALL_OF_US} Researcher Workbench Google account. This will keep your Workbench browser sessions isolated if you use other Google accounts.
